# Changelog

All notable changes to this project will be documented in this file.

## [v1.2.0] - 2026-04-04

### 新功能

- **域名路由支持** - 添加路由时支持域名格式（如 github.com），VPN 连接时自动解析为 IP 地址
- **VPN 显示控制** - 启用/禁用控制自动路由，隐藏/显示控制列表展示
- **开机启动** - 支持设置开机自动启动，一键开关无需手动授权
- **设置页面** - 全新的设置界面，分组管理权限、应用、隐藏 VPN 等

### 改进

- **禁用 VPN 视觉优化** - 禁用状态显示暂停图标和橙色标签，更直观区分
- **VPN 列表** - 右侧添加隐藏按钮，无需右键菜单
- **拖拽排序** - 目标位置元素高亮显示，更清晰的操作反馈
- **代码架构** - 拆分 VPNMonitor actor，AppController 精简至 458 行

### 修复

- 修复向下拖拽排序位置不正确的问题
- 修复 VPN 配置更新后界面不刷新的问题

## [v1.1.0] - 2026-04-03

### 新功能

- **路由规则排序** - 支持拖拽调整路由规则的顺序，按住左侧手柄图标 (≡) 上下拖动
- **路由规则备注** - 为每个路由规则添加备注说明，方便识别用途
- **实时拖拽效果** - 拖拽时其他行自动让位，直观显示目标位置

### 改进

- 使用 SCDynamicStore 替换 NWPathMonitor 监听 VPN 状态，更稳定可靠
- 移除自定义接口功能（简化配置，自动检测更可靠）

### 修复

- 修复 SCDynamicStore 监听偶发性失效问题
- 修复拖拽排序后数据未正确保存的问题

## [v1.0.0] - Initial Release

- 自动检测 VPN 连接状态
- 为每个 VPN 配置独立路由规则
- VPN 连接时自动添加路由
- 支持 L2TP/IPSec 和其他 VPN 类型
- 免密授权配置（可选）
- 菜单栏快速操作
