# Changelog

All notable changes to this project will be documented in this file.

## [v1.5.0] - 2026-04-04

### 新功能

- **公网 IP 查询** - 查询当前公网 IP 及地理位置、运营商信息
- **路由表查看** - 查看系统路由表，支持按接口和 IP 过滤
- **路由追踪** - 追踪到目标的网络路径，实时显示每一跳

## [v1.4.0] - 2026-04-04

### 新功能

- **Toast 提示** - 操作反馈更直观，支持成功/警告/错误等类型
- **路由查询工具** - 查询 IP 或域名走哪个网卡出口，支持域名解析

### 改进

- **VPN 列表交互** - 点击任意位置均可选中，禁用状态显示橙色选中效果
- **路由列表滚动** - 超过数量后可滚动，不再无限增高
- **重复路由检测** - 添加路由时自动检测是否已存在

### 修复

- 禁用 VPN 后路由配置列表消失的问题
- "已禁用"和"已连接"标签位置跳动的问题

## [v1.3.1] - 2026-04-04

### 改进

- **一键启用** - 首页直接授权安装后台服务，无需跳转设置页面
- **一次授权** - 免密授权和守护进程安装合并为一次授权操作
- **清理优化** - 卸载守护进程时同时清理免密授权配置

### 修复

- SCDynamicStore 服务名称警告

## [v1.3.0] - 2026-04-04

### 新功能

- **后台服务** - 守护进程统一处理自动路由，退出应用后仍生效
- **守护进程控制** - 设置页面新增启动/停止按钮

### 改进

- **架构优化** - GUI 与守护进程职责分离，无冲突
- **安装引导** - 主界面醒目提示安装守护进程
- **日志位置** - 守护进程日志统一到 `~/Library/Logs/RoutePilot/`
- **版本显示** - 动态读取版本号，不再硬编码
- **错误重试** - 守护进程 VPN 名称获取增加重试机制

### 修复

- AppleScript 引号转义问题

## [v1.2.0] - 2026-04-04

### 新功能

- **域名路由** - 支持 `github.com` 等域名格式，自动解析 IP 地址
- **VPN 控制** - 启用/禁用控制自动路由，隐藏不常用的 VPN
- **开机启动** - 一键开关，无需手动授权
- **设置页面** - 全新界面，分组管理

### 改进

- **拖拽排序** - 目标位置高亮显示，操作更直观
- **禁用 VPN 视觉优化** - 禁用状态显示暂停图标

### 修复

- 拖拽排序位置不正确
- VPN 配置更新后界面不刷新

## [v1.1.0] - 2026-04-03

### 新功能

- 路由规则拖拽排序
- 路由规则备注说明

### 改进

- SCDynamicStore 替换 NWPathMonitor，更稳定

### 修复

- SCDynamicStore 监听偶发失效
- 拖拽排序数据未保存

## [v1.0.0] - Initial Release

- 自动检测 VPN 连接状态
- 独立配置每个 VPN 的路由规则
- VPN 连接时自动添加路由
- 免密授权配置
- 菜单栏快速操作
