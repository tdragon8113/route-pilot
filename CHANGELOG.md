# Changelog

All notable changes to this project will be documented in this file.

## [v1.3.0] - 2026-04-04

### 新功能

- **后台服务** - 守护进程统一处理自动路由，退出应用后仍生效

### 改进

- **架构优化** - GUI 与守护进程职责分离，无冲突
- **安装引导** - 主界面醒目提示安装守护进程

### 修复

- AppleScript 引号转义问题

## [v1.2.0] - 2026-04-04

### 新功能

- **域名路由** - 支持 `github.com` 等域名格式，自动解析 IP 地址
- **VPN 控制** - 启用/禁用控制自动路由，隐藏不常用的 VPN
- **开机启动** - 一键开关，无需手动授权
- **设置页面** - 全新界面，分组管理

### 改进

- **拖拽排序** - 目标位置高亮显示，操作更直观
- **禁用 VPN 视觉优化** - 禁用状态显示暂停图标

### 修复

- 拖拽排序位置不正确
- VPN 配置更新后界面不刷新

## [v1.1.0] - 2026-04-03

### 新功能

- 路由规则拖拽排序
- 路由规则备注说明

### 改进

- SCDynamicStore 替换 NWPathMonitor，更稳定

### 修复

- SCDynamicStore 监听偶发失效
- 拖拽排序数据未保存

## [v1.0.0] - Initial Release

- 自动检测 VPN 连接状态
- 独立配置每个 VPN 的路由规则
- VPN 连接时自动添加路由
- 免密授权配置
- 菜单栏快速操作
